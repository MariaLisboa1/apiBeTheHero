# apiBeTheHero
 API for the Be The Hero project

## Getting Started

clone this repository: `git clone https://github.com/MariaLisboa1/apiBeTheHero.git`

### Prerequisites

1 - Node.js 10.9.0 or later

2 - Npm package manager

### Installing

1 - `npm install`

### Run the Server

1 - `npm start`
